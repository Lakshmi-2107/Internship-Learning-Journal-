## 🚀 Key Learnings So Far
During Week 1, I successfully set up a complete Linux-based development environment on Windows using WSL and Ubuntu. I learned how to work confidently in the terminal, navigate the Linux file system, and manage system packages using apt.

I configured Python projects using pyenv and UV for version and dependency management, and practiced Git and GitHub workflows such as creating repositories, cloning, committing, pushing code, and setting up SSH authentication.

I also explored modern development tools like VS Code and LLM CLI to improve productivity and interact with AI models directly from the command line.

Overall, I am now comfortable using Linux, Git, Python tools, and terminal-based workflows for efficient and professional software development.

---
During Week 2, I developed a strong understanding of containerization, backend development, automation, and deployment workflows. I learned how to use Podman to run applications inside containers, manage volumes and networks, and isolate environments for consistent development.

I built and tested REST APIs using FastAPI, handled routing, validation, and file uploads, and practiced testing endpoints using curl and automated scripts. I also containerized applications using Docker and understood how to deploy them to cloud platforms.

Additionally, I explored GitHub Actions for workflow automation, implemented CI/CD concepts, hosted static sites using GitHub Pages, and deployed AI applications using modern cloud tools. I gained practical experience managing environment variables securely and debugging real-world deployment issues.

Overall, I am now confident in building, containerizing, automating, and deploying modern backend applications using industry-standard tools and workflows.

---
During Week 3, I developed a strong understanding of how modern AI systems work, including LLMs, tokenization, embeddings, and semantic search. I learned how models generate outputs through token prediction and how embeddings enable meaning-based retrieval instead of simple keyword matching.

I built systems using Retrieval-Augmented Generation (RAG), applied cosine similarity for ranking results, and implemented chunking strategies for handling large datasets. I also explored multimodal AI capabilities, including processing both text and images within applications.

Additionally, I worked with structured outputs, schemas, and function calling to integrate AI into automated workflows. I gained hands-on experience with API integration, cost optimization, and evaluation techniques for improving system reliability.

Overall, I am now confident in designing and building intelligent AI-powered systems that are scalable, reliable, and production-ready.

---

During Week 4, I focused on data sourcing and learned how to collect data from websites, APIs, and PDFs using practical techniques. I built web scraping pipelines using tools like HTTPX, BeautifulSoup, lxml, and XPath to extract structured information from large datasets.

I implemented caching to improve efficiency, reduce redundant downloads, and enable resumable workflows. I also structured extracted data into formats like JSON and CSV, making it ready for analysis and integration into AI systems.

Additionally, I explored alternative data extraction methods such as Excel web connectors and PDF table extraction using Tabula. I also practiced debugging techniques to identify and fix issues during data collection.

Overall, I am now confident in building data pipelines that acquire, process, and structure data from diverse sources for real-world applications.

---
During Week 5, I focused on preparing data by cleaning and transforming raw datasets to make them suitable for analysis. I handled issues such as missing values, duplicates, inconsistent formats, and hidden characters using tools like Excel, OpenRefine, and automated profiling libraries.

I applied spreadsheet techniques like TRIM, Find & Replace, and pivot tables, and used OpenRefine for clustering and standardizing inconsistent data. I also worked with command-line tools like grep, sort, uniq, and wc to process large datasets efficiently.

Additionally, I accessed data through APIs, processed JSON responses, and handled image and media files using tools like Pillow and FFmpeg. I also used modern editors like VS Code for quick data cleaning and transformations.

Overall, I am now confident in cleaning, transforming, and preparing high-quality datasets that are ready for analysis and AI workflows.

---
During Week 6, I focused on analyzing data using tools like Pandas, DuckDB, and visualization techniques. I performed exploratory data analysis (EDA) to understand data structure, distributions, and relationships, and worked with large datasets efficiently.

I learned key statistical concepts such as correlation, R², p-values, and outlier detection using methods like IQR. I also applied different analytical approaches depending on the problem, including trend analysis and time series methods.

Additionally, I explored geospatial analysis for real-world use cases like market planning and location analysis, and improved performance by using efficient formats like Parquet and tools like DuckDB. I also developed a habit of validating results and questioning assumptions.

Overall, I am now confident in analyzing data, applying statistical thinking, and generating meaningful, actionable insights.

---

During Week 7, I focused on data visualization and storytelling, learning how to communicate insights effectively using charts, patterns, and narratives. I explored different levels of analysis, from basic summaries to advanced techniques like heatmaps, correlation matrices, and network analysis.

I worked with visualization tools such as Seaborn, Flourish, RAWGraphs, and Kumu to create static and interactive visuals. I also learned how to use animation and dynamic elements to represent changes over time and improve interpretability.

Additionally, I explored storytelling techniques by combining visuals, text, and automated insights, and used modern tools like marimo to ensure reproducible workflows. I learned how to present complex data in a clear and engaging way.

Overall, I am now confident in transforming data into compelling visual stories that effectively communicate insights and support decision-making.
