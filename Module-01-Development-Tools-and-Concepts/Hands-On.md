In this session, I practically installed and tested WSL and Ubuntu.

## Commands Executed

Checked WSL version:

```powershell
wsl --version
```

Installed WSL:

```powershell
wsl --install
```

Listed distributions:

```powershell
wsl --list --online
```

Installed Ubuntu:

```powershell
wsl --install ubuntu-24.04
```

Started Ubuntu:

```powershell
wsl
```

Updated packages inside Ubuntu:

```bash
sudo apt update
sudo apt upgrade -y
```

- WSL installed successfully
- Ubuntu launched correctly
