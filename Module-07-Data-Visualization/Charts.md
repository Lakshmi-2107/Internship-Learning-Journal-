# 📊 Charts

## 📌 Overview
Charts are essential tools for visualizing data and identifying patterns, trends, and relationships. Tools like Excel, Seaborn, Google Charts, and RAWGraphs help transform raw data into meaningful insights.

---

## 📈 1. Time Series Charts (Excel)
Time series charts visualize how data changes over time.

### 🔹 Types
- **Line Charts** → Show trends (e.g., stock prices)
- **Sparklines** → Mini in-cell charts for quick comparisons
- **Pivot Charts** → Summarized insights from large datasets

### 🔹 Key Metrics
- **Average (Mean)** → Central value
- **Standard Deviation** → Measures volatility
- **Forecasting** → Using `GROWTH()` for prediction

---

## 🔗 2. Scatter Plots & Correlation
Used to analyze relationships between variables.

- Identify **positive/negative correlation**
- Add **trendlines**
- Use **R² (R-squared)** for strength
- Detect **outliers**

---

## 📊 3. Distribution & Statistical Charts (Seaborn)

- **Distplot** → Histogram + KDE
- **Joint Plot** → Compare 2 variables
- **Pair Plot** → Full dataset relationships

---

## 📂 4. Categorical Charts

- **Bar Chart** → Category comparison
- **Count Plot** → Frequency count
- **Box Plot** → Quartiles & outliers
- **Violin Plot** → Density + distribution
- **Swarm/Strip Plot** → Non-overlapping scatter

---

## 🌡️ 5. Matrix & Heatmaps

- **Heatmaps** → Correlation using colors
- **Cluster Maps** → Group similar variables

---

## 🌐 6. Web-based Charts (Google Charts)

### 🔹 Types
- Pie Chart
- Bar Chart
- Line Chart
- Column Chart

### 🔹 Features
- Interactive (hover, tooltips)
- Customizable with CSS
- Dynamic data updates

---

## 🧠 7. Advanced Visualizations (RAWGraphs)

- Multi-dimensional mapping:
  - X-axis, Y-axis
  - Size (e.g., revenue)
  - Color (e.g., category)
- Export as SVG/PNG
- No coding required

---

## ✅ Key Takeaway
Charts are not just visuals—they:
- Reveal trends  
- Show relationships  
- Support decision-making
