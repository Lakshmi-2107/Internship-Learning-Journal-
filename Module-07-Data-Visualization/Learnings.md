## 🧱 1. Data Analysis Levels

### 🔹 Level 1: Basic
- Time series charts
- Averages & volatility
- Forecasting

### 🔹 Level 2: Relationships
- Scatter plots
- Correlation (R²)
- Outliers

### 🔹 Level 3: Advanced
- Correlation matrix
- Heatmaps
- Pattern discovery

## 🧹 2. Data Cleaning

### 🔹 Common Issues
- Wrong data types
- Hidden characters (`\n`)
- Missing values

### 🔹 Solutions
- Detect using string functions
- Filter bad rows
- Convert types properly

📌 **Lesson:** Cleaning often takes more time than analysis

---

## ⚙️ 3. Efficient Data Handling

- Use **Sparse Matrices (CSR)**
- Reduce memory usage
- Apply matrix multiplication
